# 🧪 Teste — Ambiente de Desenvolvimento Tailwind CSS

Projeto de sandbox para explorar e testar configurações do **Tailwind CSS** com **PostCSS** durante a graduação em Licenciatura em Informática na **UTFPR**.

## 📄 Sobre

Ambiente simples criado para experimentar o pipeline de build do Tailwind CSS, testando classes utilitárias e configurações antes de aplicá-las em projetos maiores.

## 🚀 Como executar

```bash
npm install
npm run build
```

Abra o `index.html` no navegador.
